Introduction

Device Manager is the administrative interface to devices in the Printing Lab. Through this web application, users can manage existing devices and send printing jobs to 3D printer and monitor/control the printing tasks.

Main features of Device Manager are as follows:

User Events

  • View events (published and pending)
  • Manage events (create, view, edit, delete, and publish)
  • Send notifications

Calendar Events

  • View event sources
  • View events (published and pending)
  • Manage events (publish, make pending, and delete)
  • Manage calendars (add and approve)
  • Schedule daily downloads
